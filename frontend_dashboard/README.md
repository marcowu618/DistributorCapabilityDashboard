# 前端看板模板（低代码平台套壳版）

## 目录
- index.html: 页面结构
- styles.css: 页面样式
- app.js: 数据获取、计算逻辑、图表渲染

## 数据来源
当前默认读取同目录下的 `data.json`（由 `评分详表` 转换而来），字段需包含：
- 区域, 分公司, 渠道名称, 应用专员, 能力评级
- 8个技术维度: 性能验证能力, 无忧切换能力, 结果问题解决能力, 风险管理能力, IQC/EQA管理能力, 瑞印课堂-系列课能力, 产品价值优势传递能力, 学术支持&文章合作能力
- 4个软技能维度: 工作态度与责任心, 学习成长能力, 沟通协助能力, 执行力与结果产出

## 本地预览
直接打开 `index.html` 即可预览（默认读取 `data.json`）。
如果是 `file://` 直接打开，页面会自动回退读取 `data.inline.js`。

## 数据更新（推荐流程）
1. 用最新 Excel（子表 `评分详表`）重新生成 `data.json`
2. 同步生成 `data.inline.js`（供 `file://` 直开时使用）

## 发布到低代码平台
1. 在平台创建一个“自定义页面/HTML容器”
2. 将 `index.html + app.js + styles.css` 放到平台静态资源目录
3. 页面入口指向 `index.html`
4. 如果改成平台接口联调，只改 `app.js` 里的 `loadData()`

## 备注
- 当前评分合计计算逻辑与原 `app.py` 对齐：
  - 专业技术能力 = 8个技术维度平均值
  - 评分合计 = 专业技术能力*4 + 工作态度*1 + 学习成长*2 + 沟通协助*1.5 + 执行力*1.5
